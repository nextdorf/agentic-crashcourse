# Repository Instructions

- This is currently a Markdown-only crash course repo; there are no verified build, test, lint, typecheck, or package-manager commands.
- Course chapters live in `crashcourse/`. Existing chapter files are scaffolds and may intentionally be incomplete.
- `local/` is for user-local notes and scratch material. It is gitignored except for `.gitignore` and `.gitkeep`; do not treat files there as publishable course content unless the user explicitly asks.
- `plans/` exists but is currently empty.
- Repo-local OpenCode configuration is currently in `.opencode/opencode.json`; it enables the Context7 MCP server and broadly allows common read/search/edit/task/web tools.
- When writing course material, prefer compact practical workshop content over exhaustive background. Move broad surveys, niche comparisons, provider opinions, and deep rabbit holes to appendix or further reading material.
